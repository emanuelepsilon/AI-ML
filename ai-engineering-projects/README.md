# AI Engineering Internship Projects

This folder is a practical learning track for applied AI internship skills: LLM APIs, RAG, embeddings, vector databases, agents, evaluations, pipelines, MCP, deployment, logging, privacy and portfolio presentation.

Start here:

1. Read `ROADMAP.md`.
2. Build `01_llm_api_chat`.
3. Move to RAG, vector search, agents, evaluations and a final applied prototype.

Recommended first three builds:

1. `01_llm_api_chat`: learn how a Python app calls an LLM API.
2. `02_document_rag_assistant`: learn chunking, embeddings, retrieval and grounded answers.
3. `03_vector_search_lab`: learn semantic search, metadata filters and vector database behavior.
