- Multi-step task evaluation
- Refusal and uncertainty tests
- Regression test suite
- Failure-case gallery

## Phase 5: Portfolio-Ready Interface

- Interactive dashboard
- Query replay
- Source inspection
- Retrieval timeline
- Benchmark summary page
