# Edge AI Notes

Deploying neural networks to constrained edge hardware requires measuring accuracy, latency, memory usage, model size, and energy consumption. Optimization can include quantization, pruning, operator selection, architecture changes, and conversion to deployment formats such as TensorFlow Lite.

Good edge AI evaluation compares the model before and after optimization. The important result is not only smaller size or faster inference, but the tradeoff between efficiency and predictive performance.
